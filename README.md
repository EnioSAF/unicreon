
# Unicreon — FoundryVTT v12 system (0.1.0)

Système non-officiel pour jouer à **Unicreon** dans Foundry VTT 12.
Les règles d'Unicreon sont sous **CC BY 4.0** (crédits: Amaury Bouchard / Rolis).

## Installation
- Dézippez ce dossier dans `Data/systems/unicreon/`, ou utilisez le zip fourni.
- Choisissez le système **Unicreon** dans la configuration du Monde.
- Créez un acteur de type "personnage".

## Fonctionnalités
- Fiche acteur (caracs d4–d12, compétences-avec-dé, PV/PK/PO).
- Bouton **Lancer un test** : carac + comp, **garde le meilleur**, difficulté 3/4/5/6/8/10, karma qui **baisse la difficulté**.
- Objets/Items (armes, armures, objet à tag d'effet, potions, incantations, pouvoirs).
- Cartes de chat de résultat.
